Playlist

A Propos du projet:

C'est une application web communicante, en l'occurrence un mini site web dynamique en PHP / HTML / CSS, de 2 ou 3 pages, affichant la playlist d'un utilisateur sélectionné dans une liste déroulante, playlist stockée dans une base de données) dans un environnement réseau, que l’on doit simuler dans un environnement Marionnet.


Infrastructure:

  • Une machine DHCP pour configurer les machines présentes dans l’infrastructure.
  • Une machine hébergeant le code source de notre application.
  • Une machine hébergeant la base de données.
  • Trois machines clientes afin d’accéder au sous-réseau.
  • Des commutateurs.
  • Des routeurs.
  • Un gateway.
  • Une machine qui sert de pare-feu.
  • Une machine avec la backup.


Nous aurons donc au total, 8 machines. On utilisera MySQL comme système de gestion de base de données avec SQL comme langage. Pour lancer les serveurs et permettre l'exécution du php, on utilisera Apache Server. On utilisera le DHCP pour configurer les machines.


Compétences visées

Maîtriser les concepts de réseaux, de l'adressage et du configuration des services déployés

Langages utilisés:
  • Shell
  • SQL
  • PHP
